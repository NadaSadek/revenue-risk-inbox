# Revenue-Risk Inbox

An AI-assisted review queue for identifying revenue-risk signals in customer and support messages.

The goal of this project is to explore how AI can turn messy customer messages into structured, reviewable product signals: billing issues, access problems, cancellation risk, refund requests, enterprise sales intent, product bugs, and account issues.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
